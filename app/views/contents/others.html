AWS用語解説

目次

EFS

EFSはシンプルでサーバーレスのファイルシステムです。アプリケーションを停止することなくファイル追加や削除に応じて自動的に拡張/縮小できたり、複数のインスタンスから同時アクセスが可能という特徴があります。

S3 Glacier

S3 Glacier は、データのアーカイブと長期バックアップ用のストレージサービスです。長期間コスト効率よくデータを保存でき、ストレージの運用と規模拡大/縮小に伴う管理作業を AWS側が行うので、容量計画、ハードウェアプロビジョニング、データレプリケーション等の管理が不要です。

Storage Gateway

AWS Storage Gateway は、オンプレミスとクラウドベースのストレージを接続し、お客様のオンプレミスの IT 環境と AWS のストレージインフラストラクチャとの間にデータセキュリティ機能を実現するサービスです。このサービスを使用すると、AWS クラウドにデータを保存し、データのセキュリティを維持するために役立つ、スケーラブルで費用効率が高いストレージを利用できます。

DynamoDB

Amazon DynamoDB は、完全マネージド型の NoSQL データベースサービス高速で質の高いパフォーマンスとデータ容量が無制限が特徴です。

ElastiCache

ElastiCacheは完全マネージド型のインメモリデータストアです。大量のデータを扱うアプリケーションを構築したり、既存のWEBアプリケーションのパフォーマンス向上が可能です。

Database Migration Service

Database Migration Serviceはデータベース間でデータを移行できるサービスです。オンプレミスからAWS間、またはAWSからAWSの間で、データベースのデータを継続的に移行できます。

Server Migration Service

Server Migration ServiceはオンプレミスのサーバーをAWSに移行できるサービスです。大規模なサーバーの移行作業を従来よりも簡単かつ短時間が実行することが可能となりました。

CloudFront

Cloud Frontとは世界150か所以上あるエッジロケーションを使い、低レイテンシーで世界中に安全にコンテンツを配信する高速コンテンツ配信ネットワークサービスです。 最先端のセキュリティ機能を持ち、サーバーから直接配信せずCloudFrontを介してユーザーに配信することでより早く安全に配信することができます。

Direct Connect

AWS Direct Connect とはAWSとお客様のデータセンター、オフィス、またはコロケーション環境との間にプライベートネットワーク接続を確立することができます。インターネットベースの接続と比較して、低コストで高セキュリティなサービスを実現できます。

CodeCommit

CodeCommitは完全マネージド型のソース管理サービスです。セキュリティと拡張性が高く、チーム間で共同開発を円滑に進めることが可能です。

CodeBuild

CodeBuild はクラウドで動作する、完全マネージド型のビルドサービスです。ソースコードをコンパイルし、単体テストを実行して、すぐにデプロイが可能となります。CodeBuildを使うことでビルドサーバーの管理、スケーリングが不要になるというメリットもあります。

CodeDeploy

AWS CodeDeployは、さまざまなコンピューティングサービスへのソフトウェアのデプロイを自動化する、完全マネージド型のサービスです。AWS CodeDeploy を使用すると、新しい機能をすばやく簡単にリリースできアプリケーションデプロイ時のダウンタイムを回避します。またデプロイを自動化できるため、手動によるミスがなくなります。

CodePipeline

AWS CodePipeline は完全マネージド型の継続的デリバリーサービスです。アプリケーションとインフラストラクチャのアップデートのためのパイプラインのリリースを自動化します。これにより、機能とアップデートを素早く、高い信頼性で配信できます。

Cloud9

Cloud9とはクラウドベースのIDE（統合開発環境）です。 統合開発環境とは、テキストエディタ、コンパイラ、デバッガなどソフトウェア開発に欠かせないツールを1つの環境で統一的に扱えるようにしたものです。 ウェブブラウザを通じてアクセスでき複数のプログラミング言語がサポートされておりソフトウェアをクラウドにリリースするのに役立ちます。

CloudShell

Cloud Shell は、セットアップすることなくブラウザからアクセスできる、オンラインの開発及び運用環境です。コマンドラインインターフェイスから AWS のサービスを管理するために使用できます。

AWS Organizations

AWS Organizationsは環境を一元的に管理することができます。AWS アカウントを作成しリソースを割り当てたり、グループ化してワークフローを整理ができます。また、すべてのアカウントに単一の支払い方法を利用することで請求を簡素化が可能です。

CloudWatch

Amazon CloudWatch は、AWSリソースと、AWSでリアルタイムに実行されるアプリケーションの監視サービスです。CloudWatch を使用してリソースの稼働状況、CPUやメモリの使用状況、利用料金の変化などを監視することができます

AWS Auto Scaling

AWS Auto Scalingは、安定した予測可能なパフォーマンスを低コストで維持するためにアプリケーションをモニタリングし、容量を自動で調整します。

CloudFormation

AWS CloudFormation は AWSリソースのテンプレート化および環境構築に役立つサービスです。使用するすべての AWSリソースを、コード化しテンプレートを作成すれば、それをもとに次に新しい環境構築する際の時間を短縮することができます。またテンプレートを一括管理するためインフラの管理と制御における負担を削減できます。

CloudTrail

AWS CloudTrail は、AWS アカウントのガバナンス、コンプライアンス、および運用とリスクの監査を行えるようにする AWS のサービスです。ユーザー、または AWS のサービスによって実行されたアクションは、CloudTrail に記録され、いつ誰が何を実行したのかが明確になります。

Systems Manager

Systems ManagerはAWS でインフラストラクチャを表示および制御するために使用できるサービスです。Systems Manager を使用すると、複数の AWS のサービスのオペレーションデータを表示して、AWS間でタスクを自動化することができます。

Trusted Advisor

AWS Trusted Advisor は、AWS のベストプラクティスに従うためのガイダンスを提供するフルマネージドサービスです。サービスをモニタリングし、セキュリティとパフォーマンスを向上し、総コストを削減します。

AWS Well-Architected Tool

AWS Well-Architected Tool は、クラウドアーキテクトがアプリケーションやワークロード向けに安全で高いパフォーマンス、障害耐性を備えた、効率的なインフラストラクチャの構築をサポートする目的で開発されました。AWS Well-Architected では、5 つの柱（優れた運用効率、セキュリティ、信頼性、パフォーマンス効率、コストの最適化）に基づいてアーキテクチャを評価します。

Athena

Athenaは、Amazon S3内のデータを直接分析するクエリサービスです。 データの保存先の Amazon S3に対してSQL を使用してクエリの実行をすることで集計やソートなどの分析を実行することができます。

Amazon Redshift

Amazon Redshift は、クラウド内でのフルマネージド型のデータウェアハウスサービスです。数百ギガバイトのデータから開始して、ペタバイト以上まで拡張できます。これにより、大規模なデータ分析可能になります。

EMR

Amazon EMR は、ビッグデータフレームワークを使用し AWS 膨大な量のデータを処理および分析します。さらに、Amazon EMR を使用して、大量のデータを変換しAmazon S3 や DynamoDB などの他のAWS データストアやデータベースに移動することができます。

Kinesis

Amazon Kinesisはスケーラブルで持続的なリアルタイムのデータストリーミングサービスです。あらゆる規模のストリーミングデータをコスト効率良く処理するための主要機能を提供します。 Amazon Kinesis を使用すると、機械学習、分析、その他のアプリケーションに用いる動画、音声、アプリケーションログなどをリアルタイムで取り込めます。

Data Pipeline

AWS Data Pipeline は、オンプレミスやAWSのデータの移動と変換を自動化するために使用するサービスです。Data Pipelineを使用すると、保存場所にあるお客様のデータに定期的にアクセスし、そのスケールで変換と処理を行い、その結果をAWS サービスに効率的に転送できます。

IAM

AWS リソースへのアクセスを安全に管理するためのウェブサービスです。IAM を使用して、AWSの利用者や、AWSのサービスがアクセスできる範囲を制御することができます。

Cognito

Cognitoはウェブアプリケーションおよびモバイルアプリに素早く簡単にユーザーのサインアップ/サインイン、アクセスコントロールの機能を追加できるサービスです。

Secrets Manager

Secrets Managerはアプリケーション、サービス、IT リソースへのアクセスに必要な認証情報の保護をします。ユーザーとアプリケーションは、Secrets Manager API を呼び出して情報を取得しますので、秘密情報をテキストで書き込む必要がなくなります。

Certificate Manager

AWS Certificate Manager は、AWS のサービスとお客様の内部接続リソースで使用するパブリックとプライベートの(SSL/TLS) 証明書のプロビジョニング、管理、デプロイを簡単にします。AWS Certificate Manager を使用すれば、SSL/TLS 証明書の購入、アップロード、更新のプロセスを手動で行う必要がなくなります。

Key Management Service

AWS Key Management Service（AWS KMS）は、お客様のデータを暗号化するために使用される暗号化キーである カスタマーマスターキー （CMK）を簡単に作成および制御できるマネージドサービスです。

WAF & Shield

可用性、セキュリティ侵害、リソースの過剰消費に影響を与えるような、ウェブの脆弱性を利用した一般的な攻撃やボットから、ウェブアプリケーションまたは API を保護するウェブアプリケーションファイアウォールです。

Step Functions

AWS Lambda 関数および AWS の複数のサービスをアプリケーション内に簡単に配列することができるサーバーレスの関数オーケストレーターです。アプリケーションの状態を維持する、チェックポイント化されたイベント駆動型の一連のワークフローを作成および実行することができます。

Simple Notification Service

Amazon Simple Notification Service (SNS) は、クライアントに対するメッセージの配信および送信を管理するフルマネージド型のウェブサービスです。

Simple Queue Service

Amazon Simple Queue Service (SQS) は、完全マネージド型のメッセージキューイングサービスです。SQS を使用すると、あらゆる量のソフトウェアコンポーネント間でメッセージを送信、保存、受信できます。

AWS Cost Explorer

AWS Cost Explorerとはリソースの使用状況と、コストを可視化して分析できるサービスです。AWSの利用料と利用料に対するコストを常に把握することができる他に、AWSサービスの使用量、その使用量に対するコストなどのデータを分析して今後3ヶ月のコストの予測ができます。

AWS Budgets

AWS Budgets には、予算を設定して、コストまたは使用量が予算額や予算量を超えたとき、あるいは超えると予測されたときにアラートを発信できる機能です。

Amazon Connect

Amazon Connectはコンタクトセンター向けのクラウドプラットフォームです。コンタクトセンターの運用を簡素化し、数分でコンタクトセンターをセットアップできます。エージェントの効率を向上させ、コストを削減することができます。

Amazon Simple Email Service

Amazon Simple Email Serviceはユーザー自身の E メールアドレスとドメインを使用して E メールを送受信するための、簡単で費用効率の高い方法を提供する E メールプラットフォームです。導入コストが低く、低価格で高い安全性のもとE メールを送信できます。

WorkSpaces

WorkSpacesはフルマネージドの永続的なデスクトップ仮想化サービスであり、ユーザーは、サポートされている任意のデバイスから、いつでも、どこでも、必要なデータ、アプリケーションおよびリソースにアクセスできます。Amazon WorkSpaces を使うと、Windows または Linux デスクトップのプロビジョニングが数分で行えます。
